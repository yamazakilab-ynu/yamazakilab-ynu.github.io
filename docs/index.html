材料内部における電子・スピン・フォノンの役割

物質の物性は、電子状態や原子構造が複雑に絡み合って発現しています。これまで観察が困難であった振る舞いや因果関係を可視化し、新しい物理として理解することを目指します。

材料の振る舞いや応答をダイナミクスで捉える

材料の性質は、空間だけでなく時間とともに大きく変化します。高速で現象を捉える高時間分解計測システムを活用して、材料がどのように応答し、機能を発現していくのかを明らかにします。

実験と計算をつないで、材料を理解する

実験だけ、計算だけでは分からない現象があります。両者を行き来しながら、材料の性質が生まれる仕組みを多角的に明らかにします。

データサイエンス・AIで材料研究を加速する

実験や計算から得られる大量のデータには、重要なヒントが隠れています。データサイエンスやAIを活用して、材料特性を支配する要因を見つけ出します。

機能デバイスの理解から制御へ

材料・デバイスの性質を理解することは、制御への第一歩です。機能発現のメカニズムを明らかにし、デバイス応用につながる材料設計へと展開します。

About

山崎研究室について

横浜国立大学 理工学部 材料工学EPに所属する、 物性物理・材料科学・データ科学を融合する研究室（物性情報材料学研究室）です。

当研究室では、材料内部で起こる電子状態・原子構造・欠陥・スピンなどのふるまいを、 先端計測・計算科学・データ解析（AI）を組み合わせて解き明かし、 その知見を材料設計へフィードバックすることを目指しています。

放射光分光をはじめとする先端計測により、非晶質材料や薄膜材料の原子スケール構造と機能の関係を捉え、 計算科学によってその起源を説明し、AI解析によって複雑なデータから重要な要因を抽出します。 こうした「測る・理解する・予測する」を一体化した研究スタイルで、材料のふるまいを見える化します。

対象は、電動機器の省エネルギー化に資する低損失材料、相変化機能を活かした次世代メモリスタ材料、 微小な変化を捉える高感度センシング材料などです。 動的現象についても、高時間分解計測で得られる大規模データをAIで解析し、 材料の状態推定・予測を行う材料デジタルツイン化にも挑戦しています。

計測・計算・データの循環を回しながら、材料の理解から設計指針の確立までを一貫して進めることで、 新しい材料機能を社会実装へつなげることを目指します。
